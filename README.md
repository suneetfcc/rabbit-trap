

## Development Setup
### es6 build pipeline inspired from [here](https://developers.google.com/web/shows/ttt/series-2/es2015)

prerequisites:
1. npm
2. gulp v4

### Instructions
1. Clone repo
```git clone https://github.com/suneetfcc/rabbit-trap```
1. Install dependencies
```npm install```
2. Build project
```gulp```
3. Run and watch on dev server
```gulp watch```
